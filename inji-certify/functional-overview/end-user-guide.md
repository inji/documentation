# End User Guide

This guide covers the **holder experience** when credentials are issued by a **Certify-backed issuer**.

It focuses on what an end user does in a wallet. It does not cover issuer-side setup.

### What you’ll do

* Pick an issuer.
* Authenticate and consent.
* Download the credential into a wallet.
